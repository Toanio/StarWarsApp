# StarWarsApp
Это приложение, которое поможет узнать больше о Звездных войнах

## Техонологии
- Язык: Swift
- Фреймворк: UIKit
- Архитектура: MVP
- Верстка: Snapkit
- Паттерны: Singlton, Delegate
